## 📖 О проекте
Этот репозиторий содержит интерактивное руководство по созданию масштабной модели **Амурского моста** (знаменитого «Амурского чуда»). Проект объединяет в себе историческое исследование и практическое моделирование из подручных материалов.

**🔗 Ссылка на сайт:** https://madebyvika.github.io/manualforbridge/

---

## 🛠 Технический стек
Для создания методички использовались простые и эффективные инструменты:
* **HTML5 & CSS3** — чистая верстка без тяжелых библиотек.
* **Markdown** — для документации.
* **GitHub Pages** — для мгновенной публикации в сети.

---

## 🧱 Основные этапы работы
Проект разделен на логические блоки, которые описаны на сайте:

1.  **Проектирование:** Поиск чертежей, выбор масштаба и подготовка инструментов.
2.  **Заготовка деталей:** Склеивание палочек для несущих ферм и опор.
3.  **Сборка:** Монтаж пролетов и соединение конструкций в единое целое.
4.  **Финальная отделка:** Шлифовка, покраска и установка декора (мини-поезда).
